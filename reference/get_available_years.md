# Get available years

Returns the range of years for which enrollment data is available.

## Usage

``` r
get_available_years()
```

## Value

Character vector of available years

## Examples

``` r
get_available_years()
#> $min_year
#> [1] 2006
#> 
#> $max_year
#> [1] 2025
#> 
#> $description
#> [1] "North Carolina enrollment data from NC DPI Statistical Profile. Available years: 2006-2025."
#> 
```
